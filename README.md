# MeloHub
![image](https://user-images.githubusercontent.com/87606859/219323043-48f2a2dd-1be4-4da5-8f45-09d95b4f9318.png)
## MeloHub – is a social app with multiple features
## Features
#### •	Sign up using Firebase Authentication & save user data to Firebase Database
#### •	Sign in using Firebase
#### •	Follow and Unfollow users
#### •	Post photo posts
#### •	Like post 
#### •	Comment post
#### •	View comment on posts
#### •	Deleting posts
#### •	Listen music
#### •	Search for information about music
#### •	Save your favorite music
#### •	Edit profile
#### •	Search for users
#### •	Сhat with users
## Tech Stack
#### •	Kotlin
#### •	Firebase
#### •	MVVM
#### •	Room database
#### •	Dagger2/Hilt
#### •	Last.fm API
#### •	Exoplayer
#### •	Coroutines
#### •	Flow
